# Project Management API

Backend application built with NestJS and MongoDB, designed to support collaborative project and task management for developers, designers, and analysts. The system enables team collaboration, issue tracking, workflow improvements, and efficient management of both new and existing projects.

## Features

- Task and project management
- Team collaboration support
- Issue tracking and problem resolution
- Workflow and productivity improvements
- Modular architecture
- REST API
- Scalable MongoDB database structure

## Tech Stack

- NestJS
- MongoDB
- TypeScript
- Mongoose
- JWT Authentication

## Getting Started

### Installation

```bash
npm install
```

### Run development server

```bash
npm run start:dev
```

### Build project

```bash
npm run build
```

### Run production

```bash
npm run start:prod
```
